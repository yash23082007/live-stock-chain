# LivestockChain Smart Contracts

This directory contains the Solidity smart contracts for LivestockChain. 
Currently, we are looking at **ERC-1155** or simple off-chain verification (hashing on-chain while keeping the metadata locally via IPFS/Pinata) rather than an expensive ERC-721 token per animal.
